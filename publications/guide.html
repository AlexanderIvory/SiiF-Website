HTML Guide for Content Writers

April 29, 2025

Basic HTML formatting can transform your content from plain text to professional publications

This guide shows you how to format your SiiF articles using simple HTML tags. You don't need to be a web developer to add professional formatting to your content.

Basic Text Formatting

Paragraphs

Text like this is a paragraph.

Use <p>Your text here</p> to create paragraphs. Always place each paragraph in its own set of p tags.

Bold Text

To make text bold like this, use: <strong>bold like this</strong>

Italic Text

To make text italic like this, use: <em>italic like this</em>

Headings

Use headings to organize your content into sections. The main article title is already an h1, so start with h2 for major sections.

For a heading like the "Headings" title above, use: <h2>Headings</h2>

For a smaller heading like "Italic Text" above, use: <h3>Italic Text</h3>

Lists

Bullet Points (Unordered List)

To create a bullet point list like this:

  • First item
  • Second item
  • Third item

Use this code:

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

Numbered Lists (Ordered List)

To create a numbered list like this:

  1. First item
  2. Second item
  3. Third item

Use this code:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Quotes

To create a quote block like this:

"Markets can remain irrational longer than you can remain solvent." - John Maynard Keynes

Use this code:

<blockquote>
    "Markets can remain irrational longer than you can remain solvent."
    <cite>- John Maynard Keynes</cite>
</blockquote>

Links

To create a link like this, use:

<a href="https://www.unimelb.edu.au">like this</a>

Images

To add an image:

Use this code:

<figure class="article-figure">
    <img src="IMAGE-URL-HERE" alt="Description of image" class="article-chart">
    <figcaption>Figure 1: Example chart with caption</figcaption>
</figure>

Tables

To create a table like this:

Company Industry Market Cap
Apple Technology $2.8T
Microsoft Technology $2.5T

Use this code:

<table class="article-table">
    <thead>
        <tr>
            <th>Company</th>
            <th>Industry</th>
            <th>Market Cap</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Apple</td>
            <td>Technology</td>
            <td>$2.8T</td>
        </tr>
        <tr>
            <td>Microsoft</td>
            <td>Technology</td>
            <td>$2.5T</td>
        </tr>
    </tbody>
</table>

Highlight Boxes

To create a callout box like this:

Important Information

This is how you highlight important information or key takeaways in your article.

Use this code:

<div class="callout-box">
    <h4>Important Information</h4>
    <p>This is how you highlight important information or key takeaways in your article.</p>
</div>

Strategy Cards

To create strategy cards like this:

Strategy Name

This is where you would describe an investment strategy or concept in a visually distinct card format.

Use this code:

<div class="strategy-card">
    <h3>Strategy Name</h3>
    <p>This is where you would describe an investment strategy or concept in a visually distinct card format.</p>
</div>

Conclusion Section

To create a conclusion section like this:

Summary

This is how you format a conclusion or summary section of your article, making it stand out from the main content.

Use this code:

<div class="article-conclusion">
    <h3>Summary</h3>
    <p>This is how you format a conclusion or summary section of your article, making it stand out from the main content.</p>
</div>

Need Help?

If you're having trouble with HTML formatting, contact the IT team for assistance. Remember that you can view the HTML of any existing article to see how it was created—just right-click on the page and select "View Page Source".
